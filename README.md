# Zeniva Wellness Spa Website

A premium, modern, fully responsive spa and wellness website built using React + Vite.

This website is created for **Zeniva Wellness Spa**, Whitefield, Bengaluru, and is designed by **BR30 Web Services**.

---

## Project Overview

Zeniva Wellness Spa website is built for premium spa therapies, relaxing massage services, luxury body scrubs, wellness packages, and easy appointment booking.

The project is SEO-ready, mobile-friendly, fast-loading, and production-ready.
